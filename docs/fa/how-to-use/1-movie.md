# فیلم

مدل `Movie(فیلم)` دارای یک فیلد `Link(لینک)` است. فیلد `(لینک‌ها)links` از نوع `ManyToManyField` است و می‌تواند شامل چندین لینک دانلود و اطلاعات دیگر باشد.
